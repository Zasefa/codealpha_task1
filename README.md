# Hangman Game

A simple word guessing game built using Python.

## Features

* Multiple categories like Fruits, Animals, Programming Languages....
* Random word selection
* Letter guessing system
* Lives tracking   (Lives means the number of wrong guesses a player is allowed before losing the game)
* Win and Lose conditions
* Duplicate guess detection

## Technologies Used

* Python
* Random Module

## How to Run

1. Run `file.py`
2. Guess one letter at a time
3. Find the word before your lives run out
